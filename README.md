# Collaborative Services Fluids & Batteries

Initial product-catalog storefront for hydraulic fluid, batteries, engine oil, and DEF.

## Local preview

Run `npm start`, then open `http://localhost:10000`.

## Render

- Runtime: Node
- Build command: leave blank
- Start command: `npm start`

Products are maintained in `public/app.js`. Stripe checkout can replace the request-pricing flow later.
